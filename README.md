# 臨床一融合知識樹與考題系統

臨床一融合知識樹與考題系統用於整理與複習中醫國考臨床一相關內容，整合章節綱目、考題練習、資料查詢、關係圖與方劑工具，協助使用者從章節脈絡、考點分布與題目練習三個方向建立完整複習流程。

## 功能總覽

### 導覽頁

彙整各功能入口與使用情境，適合初次使用時快速了解系統架構。

### 搜尋入口

可查詢方劑、病證、證型、症狀、章節、考題與資料節點，適合快速定位特定內容。

### 知識樹

依照資料章節建立綱目式結構，可從科目、章、節、小節逐層查看內容，並支援依指定範圍練習相關題目。

### 考題系統

提供題目練習、交卷、答案檢視與詳解閱讀，支援全題庫與指定範圍練習。

### 資料頁

集中呈現各科資料內容、章節整理、相關考題與關聯節點，適合讀書時對照使用。

### 關係圖

以圖像方式呈現章節、病證、方劑、症狀、考題與相關節點之間的連結，適合整理跨章節關係。

### 考題熱點與樞紐分析

整理高頻考點、章節分布、題幹線索與選項趨勢，適合考前抓重點與檢查薄弱範圍。

### 方劑工具

支援方劑查詢、組成檢視、類方鑑別、變方關係與方族整理，適合比較相近方劑與快速複習方劑脈絡。

## 資料範圍

- 傷寒論
- 金匱要略
- 溫病學
- 中醫證治學
- 中醫診斷學

## 建議使用流程

1. 先閱讀導覽頁，了解各功能用途。
2. 依科目進入知識樹，選擇欲複習的章節。
3. 閱讀資料頁內容，確認章節重點與相關考題。
4. 使用指定範圍出題功能進行練習。
5. 透過考題熱點與樞紐分析檢查高頻考點。
6. 使用方劑工具整理方劑組成、類方與變方關係。

## 使用注意事項

本系統定位為學習、複習與資料整理工具。考題、答案、詳解與整理內容仍建議搭配原始教材、正式公布資料與課堂筆記交叉確認。
