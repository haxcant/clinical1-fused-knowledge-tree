# 臨床一融合知識樹與考題系統

這是一個以臨床一中醫國考資料為核心的靜態前端網站，整合知識樹、PDF 綱目、考題系統、資料頁、搜尋入口與相關分析工具。

## 網站功能

### 1. 導覽頁

提供各功能入口說明，協助使用者快速理解每個頁面用途。

### 2. 搜尋入口

可搜尋方劑、病證、證型、考題、綱目與相關資料節點。

### 3. 知識樹

依照五本 PDF 的章、節、小節結構建立知識樹，支援從節點進入資料與考題。

涵蓋資料：

- 傷寒論
- 金匱要略
- 溫病學
- 中醫證治學
- 中醫診斷學

### 4. 考題系統

支援依題庫、章節、知識樹節點與 PDF 綱目範圍出題。

### 5. 資料頁

呈現節點內容、關聯資料、歷屆考題與相關知識。

### 6. 關係圖與知識圖譜

提供不同視覺化方式，協助理解章節、證型、方劑、症狀與考題之間的關係。

### 7. 考題熱點與樞紐分析

用於觀察高頻考點、章節分布與題目集中區域。

## 使用方式

打開 GitHub Pages 網址後，建議依序使用：

1. 先看導覽頁，了解各功能用途。
2. 進入知識樹，選擇欲讀的科目與章節。
3. 點選節點後，可查看資料或依該範圍出題。
4. 進入考題系統後，可開始練習題目。
5. 若要查特定內容，可使用搜尋入口。
